Case Studies

代表專案 · Highlight Projects

集團級 Landing Zone AWS ProServe 協作
上市集團雲端 Landing Zone 與雲治理 Program
Role：Program Manager & Cloud PO（代理商端）

與 AWS ProServe 原廠團隊協作，為某上市集團建立多帳號 Landing Zone 架構與 Security Baseline，涵蓋網路分段、身分權限、日誌集中與監控告警。 我負責需求盤點、里程碑規劃、R&R 協調與驗收標準定義，並延伸後續 DR 與 AI 相關專案。

Result：完成集團級雲端基礎建置與治理框架，延伸出 3+ 個獨立專案，讓雲端導入從單一專案變成長期 Roadmap。

Oracle Migration 跨國科技企業
跨國科技企業 Oracle 至 AWS 雲遷移
Role：Project Manager

負責某跨國科技企業的核心系統 Oracle 至 AWS 雲遷移專案，從現況盤點、架構設計、Cutover 計畫到 Go-Live 驗收， 協調客戶 IT、原廠顧問與內部 SA，控管時程與成本，同時維持 MSP / Migration Competency 合規要求。

Result：在可接受停機窗口內完成移轉並順利上線，降低既有維運成本，並讓後續資料平台與備援策略有更高彈性。

AWS 2-Tier / ISV GTM & Program
AWS 2-Tier 通路與 ISV 生態系產品營運
Role：Cloud Product Owner

負責 AWS 2-Tier 模式的產品與通路營運，包含新服務定位、價目與毛利設計、 並整合 SAP、Veeam、Zscaler 等 ISV 解決方案，推動 Marketplace 販售資格與 Bundled Offer。 同時規劃 Program / Funding 申請策略，提高高複雜度 Migration / DR 專案的落地率。

Result：一年內主責 16+ 個專案、耕耘 25+ 家企業客戶，實際 ARR 約 USD 3.7M，Pipeline 達成率超過公司目標。
